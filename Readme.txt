1. Install Node.js

https://nodejs.org/uk/ - stable version 16.x

2. Installing prerequisites for the project and running

For Windows:

Run cmd from the projects directory and type: 

npm install

Then:

browser-sync start --server --files "*.*"

For Mac: 

In the terminal type:

% cd YourProjectDirectory

Then:

npm install

browser-sync start --server --files "*.*"
